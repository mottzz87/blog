::: tip 介绍
这是一个win 10通用开机自启脚本<br>
:::
```
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\启动
```
**通常实现开机自启，只需将应用的快捷启动方式放入上面目录即可**

## 开始

**步骤**
### 0.x
新建txt文件，编辑写入：

```bash
rasdial 宽带名称 宽带账号 密码
```
### 1.x
保存为后缀为 `.bat`的文件

### 2.x

关闭网络, 双击运行 `.bat` 测试脚本是否生效

### 3.x
将脚本放入下面目录，重启计算机。

```
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\启动
```
